# AA Maker 実装計画

## 現在位置

現在は AA Maker の MVP 範囲確定まで完了している。

- `docs/spec.md` に画面構成と主要機能仕様を整理済み。
- Penpot の画面レイアウトを参照済み。
- `docs/data-model.md` にデータモデル設計を作成済み。
- アプリ本体の雛形は未作成。
- フレームワークは Vue 3 + TypeScript + Vite に決定済み。

次に行う作業は、アプリ雛形作成。
実装は View 先行で進める。
